# Tic Tac Toe
A simple version of the classic game tic tac toe. Built in html, css and vanilla javascript.

## How to use it
1. Clone this repo in GitHub Desktop, in your terminal or go to: https://play-tic-tac-toe-game.netlify.com/
2. Start playing!
